# Code of Conduct

## Our Pledge
We want this project to be welcoming, respectful, and useful for students, educators, and contributors. Everyone participating in this project is expected to help create a harassment-free experience for all.

## Our Standards
Examples of behavior that contributes to a positive environment include:
- Using welcoming and respectful language
- Being thoughtful when giving feedback
- Respecting different viewpoints and experiences
- Accepting constructive criticism gracefully
- Focusing on what is best for the school community and contributors

Examples of unacceptable behavior include:
- Harassment, threats, or discrimination of any kind
- Personal attacks, insults, or inflammatory comments
- Publishing private information without permission
- Disruptive, abusive, or inappropriate behavior in project spaces

## Enforcement Responsibilities
Project maintainers are responsible for clarifying and enforcing this Code of Conduct. Maintainers may remove, edit, or reject comments, commits, code, issues, or other contributions that are not aligned with this Code of Conduct.

## Scope
This Code of Conduct applies within all project spaces, including the repository, discussions, issues, and any linked community spaces related to this project.

## Enforcement
If you experience or witness unacceptable behavior, contact the project maintainer through the repository contact information or GitHub profile associated with the project.
